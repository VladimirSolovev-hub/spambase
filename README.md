В этом репозитории хранится проект 'spambase'. 
Задача, которая стояла передо мной заключалась в том, чтобы используя методы машинного обучения решить задачу классификации писем на спам/не спам.
В ходе решения была задействована полносвязная нейронная сеть, Логистическая регресия.
Проведена кросс-валидация, выбрана наиболее хорошая модель -- подроьности в ноутбуке.
